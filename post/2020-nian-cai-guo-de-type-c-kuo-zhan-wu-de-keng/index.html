2020年踩过的Type-c扩展坞的坑

2020-12-31 数码产品

按购买顺序列举：

测试的设备有：matebook x pro 2018, Thinkpad x280, Dell xps13, iPad pro 2020, MacBook Air m1

非专业评测，按实际体验情况记录。

  1. 绿联[5合1[HDMI+USB3.0*3+PD]4K60Hz]

    链接：https://item.jd.com/100016042988.html

    当时为了matebook x pro 2018黑苹果外接显示器，但是不能输出4k60hz画面，后来闲置。

    年底购入MacBook Air m1上能输出4k60hz画面。

    使用体验：

    1. 适合MacBook使用，我测试的windows机器不能输出4k60hz画面
    2. usb口不稳定，拔插后会暂时断连
    3. 便宜
    4. 发热大，适合随身带应急用

  2. 戴尔tb16雷电扩展坞


使用体验：

  1. 适合windows机器使用，闲鱼上300块左右的价格比较划算

  2. matebook黑苹果10.15能正确输出4k60hz画面，其他功能也正常；升级到big sur后不能输出4k60hz画面了。

  3. 自带风扇，发热可以忽略，适合在公司长期使用

  4. anker type-c 8合一扩展坞


使用体验：

  1. 适合MacBook、iPad pro 2020使用,我测试的windows机器不能输出4k60hz画面

  2. usb速率为10Gbps

  3. MacBook睡眠后经常点不亮外接4k显示器

  4. 发热较大，比绿联稍好，适合在家使用

  5. Thinkpad 40AC雷电扩展坞


使用体验：

  1. Macbook、Windows都可以使用

  2. MacBook睡眠后唤醒可以点亮外接4k显示器

  3. 发热还行，虽然没有风扇，但是体积大，温度还OK。适合公司长期使用

  4. 需要用Thinkpad机器先升级固件后，MacBook Air m1才能输出4k60hz画面

  5. 前置雷电可以接type-c USB 协议10Gbps固态硬盘

  6. Apple USB-C数字影音多端口转换器


使用时间不长，体验后续再补

使用体验：

  1. 输出色域422
  2. 长度较短
  3. 拼多多买非常划算

结论：

当时为了省钱，折腾着么多设备后发现还不如一步到位闲鱼买Thinkpad 40AC或者CalDigit TS3 Plus。
